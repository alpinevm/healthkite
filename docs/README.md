# HealthKite MCP docs

[Astro Starlight](https://starlight.astro.build/) site. Live at https://docs.healthkite.app/.

## Local dev

```bash
cd docs
npm install
npm run dev
```

Dev server at http://localhost:4321.

## Build

```bash
npm run build
```

Output: `dist/`. Deployed on Railway from this `docs/` directory.
