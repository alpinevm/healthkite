`status`

## Fields

| Field | Type | Notes |
| --- | --- | --- |
| `name` | string | Always `"HealthKite MCP"` |
| `version` | string | App `CFBundleShortVersionString` |
| `workoutCount` | integer | Total workouts visible via HealthKit |
| `sampleEncoding` | string | Wire-format discriminator (`"columnar-v1"`) |

`workoutCount` is the same number `list_workouts` will report under `total`.
